
INSTALLATION: 
------------------------------------------------------------

1. SDK's ( vrjuggler and osg ) are installed here.. C:\sdk
2. Source code , OpenSceneGraph based Viewer, and other app are installed here C:\src

3. Tortoise 64bit installed at default location. 
4. Java installed at default location. 

Environment Varibles: 
------------------------------------------------------------

1. Set for all users
2. List:
	Stereo
	----------------------------------------------------
	OSG_SCREEN_DISTANCE
	OSG_SCREEN_WIDTH
	OSG_SCREEN_HEIGHT
	
	For developers and users 
	----------------------------------------------------
	OSG_ROOT_DIR
	OSG_INC_DIR
	OSG_LIB_DIR
		

How to run Viewer
------------------------------------------------------------
1. Go to C:\src\ViewerI3dea
2. You should see a "run.bat" file
3. Drag and drop files you like to see on the script at 2.
